This programm creates header files for working with Nokia 5110 / 3310 LCD displays.
It converts .bmp image files in to code that contains info about pixels on the LCD.

It creates 3 files:
1) "dot matrix" Just set of 1 and 0 representing .bmp image.
2) "vertical.h", header that represents pixels on display with vertical addressing method.
3) "horizontal.h" horizontal method.
